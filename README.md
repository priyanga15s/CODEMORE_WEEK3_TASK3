# CODEMORE_WEEK3_TASK3

Create a custom exception class 'InvalidAgeException' that extends the 'Exception' class. Write a program that takes user input for age and throws 'InvalidAgeException' if the age is less than 18. This exercise will teach you how to create and use custom exception classes.
